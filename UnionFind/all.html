Question One


Union-find with specific canonical element. Add a method 𝚏𝚒𝚗𝚍() to the union-find data type so that 𝚏𝚒𝚗𝚍(𝚒) returns the largest element in the connected component containing i. The operations, 𝚞𝚗𝚒𝚘𝚗(), 𝚌𝚘𝚗𝚗𝚎𝚌𝚝𝚎𝚍(), and 𝚏𝚒𝚗𝚍() should all take logarithmic time or better.

Solution


Implement weighted quick union with an extra array recording the largest element.
Initialize three arrays root[] , size[] , and largest[] . Every index of size[] is initialize to 1, largest[i] to i, and root[i] to i.
Union: root[rootOfSmallerSize(x)] = rootOfLargerSize(y)
Connected: check if rootOf(x) == rootOf(y)
Find: return largest[rootOf(x)]
